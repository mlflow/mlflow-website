MLflow maintainers
MLflow maintainers

MLflow 3.5.0 includes several major features and improvements!

  • üéØ Flexible Prompt Optimization API: Introduced a new flexible API for prompt optimization with support for model switching and the GEPA algorithm, enabling more efficient prompt tuning with fewer rollouts. See the documentation to get started. (#18183, #18031, @TomeHirata)
